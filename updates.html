    Current: 1.0 → New: 1.3

    Last updated: September 25, 2025

    Summary of Update to Dynamic Sounds v1.3
    Key Fixes and Improvements:
    Fixed Notification Sound Playback: Resolved the issue where custom sounds weren't playing for notifications that the system normally plays sounds for. The app now plays custom sounds for all notifications (subject to app selection), ensuring compatibility with various notification types.

    Removed Progress Notification Skip: Eliminated the check that skipped progress notifications (e.g., download bars), allowing the app to process more notification types.

    New Vibration Feature: Added a dedicated "Vibration" tab in the bottom navigation bar for customizing vibration patterns per app.

    New Features:
    Custom Vibration Patterns: Users can select from predefined vibration patterns for each app:

    None
    Short
    Medium
    Long
    Pulse
    Double Pulse
    Heartbeat
    Vibration Preview: Each app's vibration setting includes a preview button to test the selected pattern.

    Per-App Customization: Vibration patterns are saved individually for each app, allowing fine-tuned control.
